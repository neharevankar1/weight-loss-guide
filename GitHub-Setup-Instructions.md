# How to Host Your Website on GitHub Pages

## Complete Setup Guide (10 minutes)

Your website "Understanding Your Body: Weight Loss Made Clear" is ready to share! Follow these steps to put it online for free.

---

## STEP 1: Create GitHub Account (2 minutes)

1. Go to: **https://github.com**
2. Click **"Sign up"** (top right)
3. Enter:
   - Email address
   - Password
   - Username (choose something simple, like: yourname-wellness)
4. Verify you're human (puzzle)
5. Check your email and verify account

**Done!** You now have a GitHub account.

---

## STEP 2: Create New Repository (2 minutes)

1. After logging in, click the **"+"** icon (top right)
2. Click **"New repository"**
3. Fill in:
   - **Repository name:** `weight-loss-guide` (must be lowercase, no spaces)
   - **Description:** "Understanding Your Body: Weight Loss Made Clear"
   - **Public** (select this - it's free!)
   - **✓ Check:** "Add a README file"
4. Click **"Create repository"**

**Done!** You have a repository (think of it as a folder for your website).

---

## STEP 3: Upload Your Website File (2 minutes)

1. You're now on your repository page
2. Click **"Add file"** button → **"Upload files"**
3. Drag and drop (or click "choose your files"):
   - **weight-loss-guide.html** (the file you just downloaded)
4. Scroll down
5. Click **"Commit changes"** (green button at bottom)

**Done!** Your file is uploaded.

---

## STEP 4: Enable GitHub Pages (2 minutes)

1. On your repository page, click **"Settings"** (tab at top)
2. Scroll down left sidebar
3. Click **"Pages"** (under "Code and automation" section)
4. Under "Branch":
   - Click dropdown that says "None"
   - Select **"main"**
   - Leave "/ (root)" selected
5. Click **"Save"**
6. **WAIT 1-2 MINUTES** (GitHub is building your site)
7. Refresh the page
8. You'll see a box that says: **"Your site is live at [URL]"**

**Your website is now LIVE!** 🎉

---

## STEP 5: Access Your Website (1 minute)

Your website URL will be:
```
https://[your-username].github.io/weight-loss-guide/weight-loss-guide.html
```

Example: If your username is "neha-wellness":
```
https://neha-wellness.github.io/weight-loss-guide/weight-loss-guide.html
```

**Copy this URL and save it!** This is what you'll share with friends.

## TROUBLESHOOTING

**Problem:** "Website shows 404 error"
**Solution:** 
- Wait 2-3 minutes after enabling Pages (it takes time to build)
- Check your URL is correct (include /weight-loss-guide.html at end)
- Refresh the page

**Problem:** "I can't find the Settings tab"
**Solution:** 
- Make sure you're on YOUR repository page (github.com/[username]/weight-loss-guide)
- Settings tab is at the top, after "Code", "Issues", "Pull requests"

**Problem:** "Upload didn't work"
**Solution:** 
- Make sure file name is: weight-loss-guide.html (not .txt or .doc)
- Try a different browser (Chrome works best)
- File size should be around 105KB

**Problem:** "Friends can't access website"
**Solution:** 
- Check repository is set to "Public" (not Private)
- Go to Settings → General → scroll to bottom → "Change repository visibility"
- Send them the FULL URL (including https:// and .html at end)
